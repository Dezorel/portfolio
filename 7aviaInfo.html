7avia

Веб-сайт предназначенный для онлайн покупок авиабилетов. Сайт разрабатывался для годового проекта в Техническом Университете Молдовы

Поставленные цели:

Сайт создан для финального проекта по курсу "Базы данных" в Техническом Университете Молдовы. Темой проекта являлось создание full-stack сайта с использованием баз данных.
Был реализован следующий функционал:
1. Покупка билета на определённое время 2. Чекин билета 3. Просмотр популярных направлений 4. Просмотр используемых компанией самолётов

Используемые технологии

HTML, CSS, Bootstrap, MySQL
