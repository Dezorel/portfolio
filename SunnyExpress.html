Sunny Express

Sunny Express сайт-сервис для предоставления пользователям возможности покупок товаров в китайских-онлайн магазинах

Поставленные цели:

1. Пользователь при покупке указывает китайский адрес компании Sunny Express.
2. В приложении, пользователь указывает номер чека и общую информацию о купленном им товаре. Важно, чтобы информация была действительной.
3. Пользователь может попросить о дополнительных услугах: фото товара, разделение посылки, распаковка и др.
4. После сбора общей посылки из нескольких товаров, посылка будет отправлена по адресу компании Sunny Express.
5. Покупатель оплачивает доставку при взятии посылки из офиса Sunny Express. С момента отправки информации о товаре на сайте Sunny Express, товар попадает в базу данных, где отслеживается его состояние. Существует несколько состоянии: -Ожидание - Данное состояние принимает товар после того, как информация о нем введена в систему.

-Получено - устанавливается после отправки информации о выполненных доп. услугах (фото, текст, видео т.д.)
-Отправлено - устанавливается вручную модератором при отправке посылки.
-Ожидает получателя/Задержано на таможне - устанавливается вручную модератором при получении товара в офисе / при получении сообщения о задержке товара на таможне.
-Оплачено - устанавливается вручную модератором при оплате посылки в офисе.
