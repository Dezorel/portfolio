Sunny Express

Sunny Express сайт-сервис для предоставления пользователям возможности покупок товаров в китайских-онлайн магазинах

Поставленные цели:

Моей задачей в данном проекте стала разработка сайта для компании SunnyExpress. SunnyExpress – компания, которая предоставляет возможность покупать товары в китайских онлайн-магазинах и доставляет их в Молдову. Передо мной поставили задачу разработать сайт по заказу, отслеживанию и редактированию посылок. Сайт должен быть разработан как одностраничное приложение (SPA – веб сайт использующий только один HTML документ как оболочку для всех остальных веб-страниц, организующий взаимодействие через динамически подгружаемые HTML,CSS, JavaScript)
